设置说明

只需几步即可开始使用 Wakapi 追踪您的编码时间
——简单又轻松！

1

安装 WakaTime 插件

第一步是下载并安装 WakaTime 插件到您的编辑器或 IDE。支持 Visual Studio Code、PyCharm、IntelliJ、Neovim 等众多编辑器——甚至支持 Chrome 或 Firefox 浏览器。

下载插件

2

更新您的配置

安装一个或多个 IDE 插件后，您需要调整主目录中的 WakaTime 配置文件。

在 Linux / macOS 上：~/.wakatime.cfg
在 Windows 上：%USERPROFILE%\.wakatime.cfg

用您喜欢的编辑器打开文件并按如下方式配置 API URL 和密钥：

[settings]
{{ if .User }} api_url = %s/api
api_key = {{ .ApiKey }}
{{ else }} api_url = https://wakapi.dev/api
api_key = <put your api key here>
{{ end }}

3

开始编码！

完成了！重启您的编辑器或 IDE，开始编码。稍后，您将看到您的编码活动出现在仪表盘上，提供详细的项目洞察和关于您喜欢的编辑器、操作系统、编程语言等的统计数据。

前往仪表盘
